# アプリ名: 成績管理アプリ


## 概要:
概要:本アプリは、学生の成績データを管理して科目別・月別で集計して可視化することができるアプリである。



## アピールポイント
- 学生を簡単に新しく追加できる

![デモ動画](gif/demo.gif)

## 動作条件: require
```
python 3.13 or higher

# python lib
Flask==3.0.3
peewee==3.17.7
```

## 使い方: usage

```
$ python app.py
# Try accessing "http://localhost:8080" in your browser.
```

    1. 学生のところをクリックし、学生名の入力
    2. TOPページに戻り、科目をクリックし任意の科目の追加（初回のみ）
    3. TOPページに戻り、成績をクリックし任意の学生と科目を選択し点数と実施月の登録を行う。
    4. その後、TOPページに戻ることでグラフが表示される
