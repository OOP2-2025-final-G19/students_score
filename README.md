# アプリ名: ｓtudents_score
> アプリ名は、プロジェクトの顔でもあるので、適切な命名を行ってください。

概要:
> 生徒の名前、科目名、成績の登録を行う簡易アプリケーション
このプラグラムは単一学年かつ三教科の成績管理のアプリケーションです。


## アピールポイント
科目別の成績の集計だけでなく、過去の平均点の推移を見ることができ、過去から現在までの成長度合いを一目でわかることができます。
フォントとテキストをピンク色に変更し、めちゃめちゃ可愛いです。
